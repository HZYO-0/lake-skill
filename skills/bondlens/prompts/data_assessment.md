# BondLens Data Assessment Prompt

评估数据充分性，决定分析深度。

## 评估维度

| 维度 | 不足 (< 30 条) | 基本 (30-100 条) | 充分 (100+ 条) |
|------|---------------|-----------------|---------------|
| 消息量 | 仅输出局部观察 | 分析但标注低置信度 | 完整分析 |
| 时间跨度 | < 1 天：单场景快照 | 1-7 天：短期模式 | 1+ 周：趋势分析 |
| 双方参与 | 只有一方：无法分析互动 | 严重不平衡：标注偏差 | 基本均衡：可信互动模式 |
| 场景多样性 | 只有闲聊：仅分析日常模式 | 有冲突：可分析冲突路径 | 冲突+修复：完整互动循环 |

## 决策矩阵

```
if 消息量 < 30:
    → 输出：局部观察 + 提示补充数据
    → 不输出：人格画像、依恋假设、互动模式

elif 消息量 30-100:
    → 输出：初步人格信号（标注低置信度）、简单互动模式
    → 标注：哪些维度因数据不足无法分析

elif 消息量 100+:
    → 输出：完整 9 层分析（Layer -1 行动卡 + Layer 0-7）
    → 每个结论标注置信度
```

## 数据质量问题

需额外标注：
- 时间戳缺失或不连续
- 一方消息占比 > 80%
- 全部为同一场景（如只有工作讨论）
- 明显的转发/复制内容
