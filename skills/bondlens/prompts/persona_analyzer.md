# BondLens Persona Analyzer（6 层结构版）

提取双方人格信号，生成 6 层结构画像。

> 借鉴 colleague-skill 的 6 层人格结构 + 标签翻译模式。
> 核心原则：抽象标签必须翻译为具体行为规则，每个维度必须有原话锚定。

---

## 6 层人格结构

### Layer 0：硬规则（最高优先级，不可违反）

从数据中提取"when X happens, then Y"格式的行为规则。

**提取方法**：
- 找到对方反复出现的行为模式（≥3 次）
- 转化为条件触发式规则
- 每条规则必须附证据 ID

**输出格式**：
```
规则 1：当 {触发条件} 时，对方会 {具体行为}
证据：{E-YYYYMMDD-NNN}
置信度：{低/中/高}

规则 2：当 {触发条件} 时，对方会 {具体行为}
证据：{E-YYYYMMDD-NNN}
置信度：{低/中/高}
```

**标签翻译示例**（抽象 → 具体）：
- "回避型依恋" → "当被问'我们什么关系'时，用转移话题或沉默回应"
- "边界感强" → "当对方连续关心超过 3 条时，会说'不用讷'来设立距离"
- "高情绪反应性" → "当收到认可时，会连发 3+ 条消息并用 [脸红][调皮] 表情"

---

### Layer 1：身份定位

对方在这段关系中如何看待自己和对方。

**提取维度**：
- 对方在这段关系中的角色认知（照顾者/被照顾者/平等伙伴）
- 对方对"我"的定位（朋友/暧昧对象/师兄/备选）
- 对方的自我价值感（自信/自卑/混合）

**输出格式**：
```
身份定位：{描述}
证据：{E-YYYYMMDD-NNN}
```

---

### Layer 2：表达风格

对方的说话方式、用词习惯、情绪表达。

**提取维度**：
- 高频词和口头禅
- 句式习惯（短句/长句/省略主语/语气词）
- 表情使用模式（类型、频率、替代文字的情况）
- 情绪表达方式（直接/间接/用表情代替）
- 网络用语/梗/缩写

**必须包含**：每个维度 1-2 条直接对话示例（不是描述）

```
差的写法：Tf 消息偏短，喜欢用表情
好的写法：
  - [2026-05-17] Tf: 可以
  - [2026-05-17] Tf: 好的
  - [2026-05-17] Tf: [撇嘴]
  54% 的消息 ≤5 字符
```

---

### Layer 3：决策模式

对方在压力/冲突/选择时的行为倾向。

**提取维度**：
- 压力下的行为选择（回避/面对/转移）
- 说"不"的方式（直接拒绝/婉拒/沉默/转移话题）
- 面对批评时的反应（解释/承认/反问/回避）
- 对不确定性的态度（接受/焦虑/回避）

**标签翻译**：
- "回避型决策" → "当面临关系定义时，选择沉默或转移话题，而不是直接讨论"
- "压力退出" → "当学业压力大时，会减少主动联系频率"

---

### Layer 4：关系行为

对方在亲密关系中的行为模式。

**提取维度**：
- 亲近行为：什么情况下会主动靠近
- 疏远行为：什么情况下会拉开距离
- 冲突行为：冲突时的第一反应
- 修复行为：冲突后如何修复（主动/被动/需要时间）

**必须包含**：每个行为 1-2 条原话示例

---

### Layer 5：边界红线

对方明确拒绝过的事和不可触碰的领域。

**提取维度**：
- 显性边界：直接说过"不"的事
- 隐性边界：用沉默/转移/已读不回表达的拒绝
- 雷区：触碰后情绪明显变化的话题/行为

**必须包含**：具体的拒绝原话

---

## 证据要求

每个信号必须：
1. 引用至少 1 个证据 ID（E-YYYYMMDD-NNN）
2. 提供置信度（低/中/高）
3. 列出反证（如有）
4. 提供替代解释

## 好坏示例对比

### 标签翻译

```
差的写法：Tf 是回避型依恋
好的写法：
  规则：当被要求明确关系时，Tf 会用转移话题或沉默回应
  证据：E-20260528-002（"保持低调感别说话"）
  置信度：中
  替代解释：可能只是当前阶段不想公开，而非回避型依恋
```

### 原话锚定

```
差的写法：Tf 在压力下会减少联系
好的写法：
  规则：当学业压力大时，Tf 会减少主动联系
  原话：[2026-05-28] Tf: 保持低调感别说话
  原话：[2024-11-01] Tf: 我在恐惧害怕的玩
  证据：E-20260528-002, E-20241101-001
```

```markdown
# {称呼} 人格画像（6 层结构）

## Layer 0：硬规则
{when X then Y 规则，每条附证据}

## Layer 1：身份定位
{关系角色认知}

## Layer 2：表达风格
{高频词、句式、表情、原话示例}

## Layer 3：决策模式
{压力行为、说"不"方式}

## Layer 4：关系行为
{亲近/疏远/冲突/修复模式，附原话}

## Layer 5：边界红线
{显性/隐性边界，附原话}

## 依恋假设
{综合 6 层得出的依恋假设 + 不确定性说明}
```
