# BondLens Persona Analyzer

提取双方人格信号，生成 Target Profile 和 Self Profile。

## Target Profile 结构（对方画像）

### 表达 DNA
- 信息密度偏好（详细 vs 简洁）
- 幽默风格（自嘲、调侃、冷幽默）
- 情绪表达方式（直接、间接、压抑）
- 称呼和亲密度标记

### 积极回应触发点
- 哪些话题/行为让对方回复更长、更快
- 对方主动开启的话题
- 对方表达开心/兴奋的信号词

### 压力触发点
- 哪些话题/行为让对方回复变短、变慢
- 对方回避的话题
- 对方表达烦躁/压力的信号词

### 边界与雷区
- 明确拒绝过的话题/行为
- 已读不回的场景
- 情绪突然变化的转折点

## Self Profile 结构（自我画像）

同上结构，但分析视角为"我"。

## 大五人格沟通信号

参考 `references/frameworks/big_five_communication_signals.md`，评估：

| 维度 | 评估指标 | 输出 |
|------|---------|------|
| 情绪反应性 | 感叹号密度、情绪词频率、回复速度波动 | 低/中低/中/中高/高 |
| 表达直接度 | 直接表达 vs 暗示/转移的比例 | 低/中低/中/中高/高 |
| 计划性 | 提前安排 vs 临时决定、承诺兑现率 | 低/中低/中/中高/高 |
| 共情合作 | 回应对方情绪、主动配合、让步频率 | 低/中低/中/中高/高 |
| 社交能量 | 主动开启话题、连续消息、深夜聊天 | 低/中低/中/中高/高 |
| 开放性 | 新话题引入、抽象讨论、好奇心表达 | 低/中低/中/中高/高 |

## 证据要求

每个信号必须：
1. 引用至少 1 个证据 ID（E-YYYYMMDD-NNN）
2. 提供置信度（低/中/高）
3. 列出反证（如有）
4. 提供替代解释
