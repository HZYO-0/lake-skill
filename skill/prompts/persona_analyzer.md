# 人格信号提取 Prompt

## 任务

从聊天记录中提取双方的人格信号。参考 colleague-skill 的 persona_analyzer，但适配 LakeSkill 的双方分析和证据方法论。

**优先级规则：用户手动提供的信息 > 聊天记录分析。有冲突时以用户信息为准，并在输出中注明。**

---

## 对方维度

### 1. 表达 DNA

**用词统计**
- 高频词（出现 3 次以上的词/短语）
- 口头禅（固定搭配）
- 专属用语（只有你们之间用的词）

**句式特征**
- 平均句长（短句 <15 字 / 中等 15-40 字 / 长句 >40 字）
- 是否爱用列表/分点
- 结论位置（开门见山 vs 先铺垫）
- 转折词使用频率

**情绪信号**
- emoji 使用习惯（无/偶尔/频繁，常用哪类）
- 标点密度（感叹号/省略号的使用）
- 正式程度（1=极度正式 5=非常口语化）

**温暖度与距离感**
- 什么时候表达温暖
- 什么时候拉远距离
- 距离感的表达方式（冷淡 / 忙碌 / 沉默 / 转移话题）

```
输出格式：
口头禅：["xxx", ...]
高频词：["xxx", ...]
句式：[描述]
emoji：[无/偶尔/频繁，类型]
温暖度：[1-5]
距离风格：[描述]
```

### 2. 情绪触发器

```
打开话匣子的时候：[...]
关闭话匣子的时候：[...]
变得防御的时候：[...]
表达亲近的时候：[...]
消失的时候：[...]
```

### 3. 冲突模式

```
冲突风格：[直接/间接/沉默/爆发]
防御机制：[解释/否认/反攻/逃避]
修复模式：[主动道歉/等对方/假装没事/需要时间]
沉默模式：[冷战多久/什么触发沉默/什么打破沉默]
边界反应：[被越界时怎么反应]
```

### 4. 记忆签名

```
印象深刻的场景：[...]
象征性元素：[...]
情感余味：[...]
```
