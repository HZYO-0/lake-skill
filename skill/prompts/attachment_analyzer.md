# 依恋假设分析 Prompt

## 任务

基于聊天记录，提取双方的依恋信号。参考 `references/frameworks/attachment_anxiety_avoidance.md`。

**这不是临床诊断。只观察沟通信号，不做人格判断。**

---

## 对方依恋信号

### 焦虑信号

观察以下行为（参考 attachment_anxiety_avoidance.md）：

- 反复寻求确认
- 回复延迟后明显焦虑
- 把沉默等同于拒绝
- 难以容忍模糊
- 冲突后紧急修复尝试

```
焦虑信号强度：[无/低/中/高]
证据：[引用具体消息，附证据 ID]
置信度：[低/中/高]
替代解释：[其他可能的原因]
```

### 回避信号

- 亲密话题升级后退缩
- 拖延或回避冲突对话
- 强调独立和空间
- 淡化情感话题
- 关系定义出现时转移话题

```
回避信号强度：[无/低/中/高]
证据：[引用具体消息，附证据 ID]
置信度：[低/中/高]
替代解释：[其他可能的原因]
```

### 安全信号

- 直接但温暖的沟通
- 能表达需求和边界
- 冲突后能修复
- 言行一致
- 愿意协商亲密和空间

```
安全信号强度：[无/低/中/高]
证据：[引用具体消息，附证据 ID]
```

## 混合模式识别

很多时候不是单纯的焦虑或回避，而是混合模式。识别：

- "关系确认需求 + 高压力话题退缩"
- "亲密渴望 + 被拒绝恐惧"
- "独立宣言 + 暗示需要关注"

```
混合模式：[描述]
证据：[引用]
置信度：[低/中/高]
```

```markdown
- 回避信号：{level}（证据：{IDs}，置信度：{confidence}）
- 安全信号：{level}（证据：{IDs}）
- 混合模式：{description}（证据：{IDs}）

### 我的依恋信号
（同上结构）

### 不确定性说明
- {每个结论的反证和替代解释}
```

## 禁止输出

- "对方是焦虑型人格"
- "对方童年有创伤"
- "对方有依恋障碍"
