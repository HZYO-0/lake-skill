# 沟通模式分析 Prompt

## 任务

从聊天记录中提取双方的沟通模式。这是 BondLens 独有的分析维度。

---

## 分析维度

### 1. 响应模式

**响应时间**
- 平均响应时间（过滤 >24h 的间隔）
- 响应时间趋势（早期 vs 近期）
- 谁更常先回复

**消息长度**
- 各方平均消息长度
- 长度趋势（越来越短 = 距离感增加？）
- 长消息 vs 短消息的场景

**开启话题**
- 谁更常开启新话题
- 开启方式（直接 / 分享日常 / 提问）
- 对方对新话题的回应率

### 2. 情绪表达模式

**正面情绪**
- 表达喜欢/关心的方式
- 亲密度信号（昵称、表情、语气词）
- 分享日常的频率和深度

**负面情绪**
- 表达不满的方式（直接 / 间接 / 沉默）
- 抱怨 vs 诉求的区别
- 情绪升级的信号

**情绪调节**
- 谁更常做情绪调节者
- 调节方式（转移话题 / 幽默 / 道歉 / 给空间）
- 调节效果

### 3. 话题模式

**话题偏好**
- 各方喜欢聊什么
- 回避什么话题
- 话题深度（表面闲聊 vs 深度交流）

**话题转换**
- 谁更常转换话题
- 转换原因（回避 / 自然过渡 / 无聊）
- 被转换话题时的反应

### 4. 边界模式

**个人边界**
- 各方如何表达边界
- 边界被试探时的反应
- 边界冲突的处理方式

**关系边界**
- 对关系定义的态度
- 对未来的讨论意愿
- 对亲密/距离的调节

---

## 输出格式

```markdown
### 响应模式
- 平均响应时间：对方 {x} 分钟，我 {y} 分钟
- 开启话题：对方 {n} 次，我 {n} 次
- 响应时间趋势：{描述}

### 情绪表达
- 正面情绪信号：{描述}（证据：{IDs}）
- 负面情绪信号：{描述}（证据：{IDs}）
- 情绪调节模式：{描述}

### 话题模式
- 话题偏好：{描述}
- 回避话题：{描述}
- 话题深度：{描述}

### 边界模式
- 个人边界：{描述}
- 关系边界：{描述}
```

---

## 注意事项

- 所有结论必须附证据 ID
- 区分"对方主动表达"和"对方被动回应"
- 注意时间趋势（模式可能随时间变化）
- 考虑上下文（工作时间 vs 休息时间）
