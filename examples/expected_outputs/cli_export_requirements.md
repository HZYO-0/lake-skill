# CLI Export Input: Required Output Elements

When given CLI-preprocessed data (digest + evidence index + session summaries), the Skill output MUST contain:

## Must Contain

- [ ] References to specific evidence IDs from the input (E001-E008)
- [ ] Use of session IDs to contextualize (S001, S003, S010)
- [ ] Acknowledgment that data was CLI-preprocessed
- [ ] Full analysis (data is sufficient)
- [ ] Confidence calibration based on evidence quality
- [ ] Specific quotes cited from evidence index

## Structural Requirements

- [ ] Analysis references the trend data from digest (回复时间上升趋势)
- [ ] Hypotheses linked to multiple evidence items
- [ ] Alternative explanations for "回避" theme evidence
- [ ] Alternative explanations for "关系压力" theme evidence

## Must NOT Contain

- [ ] Request for more raw data (CLI output is designed to be sufficient)
- [ ] Clinical diagnoses
- [ ] Certainty language
- [ ] Manipulation tactics
