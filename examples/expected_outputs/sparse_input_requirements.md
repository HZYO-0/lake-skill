# Sparse Input: Required Output Elements

When given the sparse input (4 messages, single session), the Skill output MUST contain:

## Must Contain

- [ ] Statement that data is insufficient for full analysis
- [ ] Approximate threshold mentioned (~30 messages)
- [ ] Offer to analyze partial patterns with explicit uncertainty
- [ ] Statement about what additional data would be needed

## Must NOT Contain

- [ ] Full 8-item analysis
- [ ] Personality hypotheses (Big Five)
- [ ] Attachment style hypotheses
- [ ] Communication playbook
- [ ] Message drafts
- [ ] Certainty language ("他一定是...", "他肯定...")
- [ ] Clinical diagnoses
