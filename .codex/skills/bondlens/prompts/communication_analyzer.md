# BondLens Communication Pattern Analyzer

分析双方沟通模式，产出沟通模式摘要。

## 分析维度

### 1. 响应模式

提取指标：
- **响应时间分布**：中位数、P25、P75（排除睡眠时间）
- **消息长度分布**：字符数中位数、≤5 字符占比
- **话题开启率**：谁更常发起新话题
- **连续消息率**：谁更常连发多条

输出格式：
```
[Tf] 响应时间中位数: X 分钟（P25: X, P75: X）
[Tf] 消息长度中位数: X 字符，≤5 字符占比: X%
[Zy] 响应时间中位数: X 分钟（P25: X, P75: X）
[Zy] 消息长度中位数: X 字符，≤5 字符占比: X%
话题开启: Tf X% / Zy X%
```

### 2. 情绪表达模式

识别信号词和模式：
- **高情绪词汇**：哈哈哈、啊啊啊、呜呜呜、！！！、？？？
- **表情符号使用频率和类型**
- **感叹号/问号密度**
- **大写/重复字符使用**

输出：双方情绪表达风格对比

### 3. 话题模式

分类话题类型：
- 日常分享（吃饭、天气、出行）
- 工作/学业（论文、项目、导师）
- 情感表达（关心、思念、抱怨）
- 信息交换（链接、文件、教程）
- 调侃/玩笑

输出：话题偏好雷达图（文字版）

### 4. 边界模式

识别边界信号：
- **显性边界**："不要"、"不想"、"别"
- **隐性边界**：已读不回、话题转移、简短回复
- **边界触发场景**：哪些话题/行为触发边界
- **边界后的恢复时间**

输出：边界信号表
