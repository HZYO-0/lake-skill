# BondLens Interaction Pattern Analyzer

分析双方互动循环模式。

## 正向循环识别

寻找双方互动中的良性循环：

### 模式特征
- A 的行为 → B 积极回应 → A 继续 → B 更积极
- 双方投入趋于平衡
- 情绪基调稳定或上升

### 常见正向循环

| 类型 | 触发 | 表现 | 结果 |
|------|------|------|------|
| 日常陪伴 | 分享日常 | 互相关注细节 | 持续联系 |
| 压力支持 | 一方压力 | 另一方倾听/建议 | 信任加深 |
| 秘密分享 | 分享脆弱 | 安全接收 | 亲密感增加 |
| 轻松互动 | 调侃/玩笑 | 积极回应幽默 | 亲密感增加 |

## 负向循环识别

寻找双方互动中的恶性循环：

### 模式特征
- A 的行为 → B 消极回应 → A 调整但方向错误 → B 更消极
- 双方投入失衡
- 情绪基调下降

### 常见负向循环

| 类型 | 触发 | 表现 | 结果 |
|------|------|------|------|
| 过度关心 | A 关心过多 | B 感到压力 | B 退出 |
| 模糊焦虑 | B 回复变短 | A 更焦虑 | A 更频繁联系 → B 更退出 |
| 投入不对称 | A 投入远超 B | A 期待回报 | A 失望 → 更多投入 |
| 误解循环 | A 误解 B 信号 | A 做出错误回应 | B 更困惑 |

## 冲突升级路径

追踪冲突如何升级：

```
触发事件
  → 初始反应（谁先反应？什么反应？）
    → 升级信号（冷战？指责？回避？）
      → 高点（最激烈时刻）
        → 修复/僵持
```

## 修复路径

识别冲突如何修复：

```
修复信号出现（谁先？）
  → 修复方式（道歉/转移话题/行动补偿/时间冲淡）
    → 恢复标志（回复速度恢复/话题恢复/情绪恢复）
      → 关系变化（更近/不变/更远）
```

## 关键转折点分析

识别关系中的重大转折：

1. **转折点识别**：消息量突变、话题突变、情绪基调突变
2. **转折前状态**：转折前的互动模式
3. **转折后状态**：转折后的互动模式
4. **转折原因假设**：基于证据的假设 + 替代解释
