# Symbolic Mode Policy

Astrology, tarot, fortune telling, and symbolic systems are disabled by default.

They must not be used as evidence for:

- personality
- attachment style
- romantic intention
- relationship outcome
- whether someone loves the user
- whether to reconnect

If the user explicitly requests symbolic reflection, label it clearly:

```markdown
以下内容仅作为娱乐性/象征性反思，不作为证据型关系判断，不影响人格、依恋或沟通建议。
```

Allowed uses:

- journaling prompts
- metaphorical reflection
- emotional storytelling
- self-reflection questions

Forbidden uses:

- predicting reunion
- judging hidden feelings
- overriding chat evidence
- telling the user to manipulate the other person
