3. **Attachment Hypotheses**（依恋假设）：依恋焦虑/回避/安全信号
4. **Interaction Patterns**（互动模式）：正向循环、负向循环、冲突升级路径、修复路径

## 关键发现

1. **关系演变**：从 2023 年的"师兄帮师妹做代码"逐步演变为 2025 年的"每天高频聊天"。2024-10 是关键转折点。
2. **Tf 人格特征**：回避-安全混合型依恋，重视边界和隐私，在感到安全时会表现亲密。
3. **Zy 人格特征**：焦虑-安全混合型依恋，高度主动，直接表达，在 Tf 疏离时会焦虑。
4. **当前挑战**：答辩后 Tf 疏离，需要给空间但保持存在感。
