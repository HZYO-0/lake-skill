# BondLens v3 vs v4 版本对比

**生成日期**: 2026-06-05
**对比版本**: v3.0.0 vs v4.1.0

---

## 主要改进

### 1. 新增 Layer -1: 关系行动卡

| 项目 | v3 | v4.1.0 |
|------|-----|--------|
| 行动卡 | 无 | Layer -1，报告第一屏 |
| 策略路由 | 无 | 低压稳定/适度推进/冲突修复 |
| 本周动作 | 无 | 4 条具体动作，含触发条件、证据、禁忌 |
| 踩雷复盘 | 无 | 2 条历史踩雷事件 |
| 观察信号 | 无 | 需观察 + 警惕信号 |
| 话术 | 无 | 3 个版本（稳妥/降压/更直接） |

### 2. 证据体系强化

| 项目 | v3 | v4.1.0 |
|------|-----|--------|
| 证据 ID | 无统一格式 | E-YYYYMMDD-NNN |
| 证据索引 | 无 | evidence_index.jsonl |
| 结论格式 | 自由格式 | 观察→证据→推断→置信度→反证→替代解释→建议 |
| 置信度 | 无 | 高/中/低 |
| 替代解释 | 无 | 每条结论至少 1 个 |
| 覆盖声明 | 无 | 数据范围、消息量、抽样方式 |

### 3. 输出规范

| 项目 | v3 | v4.1.0 |
|------|-----|--------|
| 报告结构 | 8 层 | 9 层（含 Layer -1） |
| 风险词检查 | 无 | 禁用"肯定/一定/离不开/吃醋/PUA" |
| 质量自检 | 无 | 证据完整性、风险词、覆盖声明、Layer 完整性 |
| Lint 工具 | 无 | report_lint.py |
| 审计工具 | 无 | evidence_audit.py |

### 4. 教练模式改进

| 项目 | v3 | v4.1.0 |
|------|-----|--------|
| 快速回答路由 | 无 | "怎么办"→行动卡, "帮我分析"→完整报告 |
| 话术版本 | 无 | 稳妥/降压/更直接 |

## 分析质量对比

| 维度 | v3 | v4.1.0 |
|------|-----|--------|
| 可操作性 | 低（只有分析，无行动建议） | 高（行动卡 + 话术 + 禁忌） |
| 可验证性 | 低（无证据引用） | 高（证据 ID + 置信度 + 替代解释） |
| 安全性 | 中（无风险词检查） | 高（风险词扫描 + 禁忌边界） |
| 可维护性 | 低（无工具支持） | 高（lint + 审计 + 索引） |
